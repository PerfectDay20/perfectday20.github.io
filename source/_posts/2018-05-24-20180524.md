## 收获
1. 比如在Spark中用printSchema看到的数据结构为包含map的array，用Java读取的时候，这些类型其实都是Scala中的，所以不能直接使用Java中的List、Map等；WrappedArray也可用Seq代替
```java
WrappedArray<scala.collection.Map<String, String>> es = row.getAs("colName");
for (Map<String, String> map : JavaConverters.seqAsJavaListConverter(es).asJava()) {
   // do somthing to Java map
}
```

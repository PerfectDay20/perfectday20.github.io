## 收获
在shell脚本中执行多个任务，希望前面任务失败时立即停止，且不想每个任务后都添加判断，可以使用
```bash
function doit {
    "$@"
    local status=$?
    if [[ ${status} -ne 0 ]]; then
      echo "exec $@ failed"
      exit ${status}
    fi
}

# 如果不需要输出解释信息可以直接使用
set -e

```

1. 在 Scala 中如果需要对一个集合先进行 filter，然后 map，可以合并为一次操作；
  a. 可以合并为 flatMap 操作，和 Spark 中一样，需要注意运用 match 的时候需要考虑所有情况，将需要过滤的部分返回 None 即可，且由于 flat 的作用，会返回 faltten 后的结果；
  b. 可以合并为 collect 操作，它接受一个 PartialFunction，所以只需要用 case 处理保留的部分；当需要返回集合嵌套集合时，采用这种方式
```scala
  println(Seq(1, 2, 3).flatMap(_ match {
    case i if i % 2 != 0 => Set(i, i + 1)
    case _ => None
  })) // List(1, 2, 3, 4)

  println(Seq(1, 2, 3).collect {
    case i if i % 2 != 0 => Set(i, i + 1)
  }) // List(Set(1, 2), Set(3, 4))
```

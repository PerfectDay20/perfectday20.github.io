1. code block 的值是最后一个语句，所以作为方法传递给 map 时，block本身并不是一个方法，而是最后一句，如
```scala
List(1, 2).map { println("Hi"); _ + 1 }
// 只会打印一个 Hi
// 等价于
println("Hi")
val f = { i:Int => i + 1 }
List(1, 2).map(f)
```

1. except 直接返回 distinct 的结果，不用在前后distinct; except all 暂不支持; dataset.union 是union all，但sql string中的union和union all全部支持
2. 导入 `spark.implicits._` 可以使用 `$"name"` 来通过列名选取列，是因为导入了一个自定义的 String Interpolator，将字符串交给`s String Interpolator`处理，然后生成新的ColumnName对象，而ColumnName类继承了Column类     
3. 同理，`Seq(1, 2, 3).toDS()` 这种创建Dataset的方式也是通过导入 spark.implicits._，然后将Seq隐式转换成DatasetHolder，然后调用其toDS方法实现的
```scala
  /**
   * Converts $"col name" into a [[Column]].
   *
   * @since 2.0.0
   */
  implicit class StringToColumn(val sc: StringContext) {
    def $(args: Any*): ColumnName = {
      new ColumnName(sc.s(args: _*))
    }
  }
```

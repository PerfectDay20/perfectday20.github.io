1. Array，Map都可以看作是PartialFunction
2. buffer，set 等的 xs += (a, b, c) 操作不是增加了一个tuple，而是加了三个参数，把+=看成方法，则括号里边的是参数更容易理解
3. 在对Dataset的多列进行相同处理时，可以使用 `foldLeft`，如
```scala
  def foo(col: String): Column = ???
  val newDS = Seq("col1", "col2", "col3").foldLeft(oldDS) { (tmpDS, col) => tmpDS.withColumn(col, foo(col)) }
  
  // 其实对一个对象进行多次操作，都可以用foldLeft代替for循环
  val m = mutable.Map(1 -> 2, 3 -> 4)
  Seq(1, 2, 3).foldLeft(m) { (m, i) => m(i) = 0; m }
```
